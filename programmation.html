Développer des Applications communicantes

J'ai développé une application client/server en java et XML sous android. Pour se faire j'ai eu à apprendre à utiliser le logiciel Android Studio ainsi que les APIs internets d'Android.,


Mon application devait supporter des technologies comme le Multithreading pour fonctionner aussi bien en tant que client et server. Supporter les portocoles TCP/UDP, mais également les sockets pour la communication entre les deux parties de l'application. Pour finir, j'ai mis en place une Authentification utilisateur pour le protocole TCP.


Quelques compétences mobilisées

  • Maitrise des protocoles de communications (TCP/UDP)
  • Virtualisation d'application et d'os (Android & Linux)
  • Programmation (python,java et C)
  • Développement d'Applications avec API et protocoles internet (java.net & java.io)
  • Sécuriser les Services (authentification)
  • Analyse et compréhension d'un cahier des charges technique
